query()

query(string $sql, array $data,  $one, string $cache_type, integer $timeout) : mixed

可以执行任何的SQL，包括SELECT/INSERT/CREATE等等，但是参数比较多

Parameters

string $sql

要执行的SQL
array $data

要赋值到SQL里面做参数绑定的数组，只支持一维数组 $param bool $one 是否只取单条记录，true为单条记录（一维数组），false为全部记录（二维数组），对于INSERT/UPDATE操作此参数无效
$one
string $cache_type

缓存类型，m所内存缓存，f所文件缓存，false为不进行缓存，默认内存memcache缓存模式
integer $timeout

缓存有效时间，默认2小时，必须$cache_type不为false时候有效

Returns

mixed —

执行的结果
