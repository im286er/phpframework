onEvent()

onEvent( $eventName,  $bean) : void

An observer object needs to be capable of receiving notifications. Therefore the observer needs to implement the onEvent method with two parameters, the event identifier specifying the current event and a message object (in RedBeanPHP this can also be a bean).

attachEventListeners()

attachEventListeners(\RedBeanPHP\Observable $observable) : void

Attaches the FUSE event listeners. Now the Model Helper will listen for CRUD events. If a CRUD event occurs it will send a signal to the model that belongs to the CRUD bean and this model will take over control from there.
