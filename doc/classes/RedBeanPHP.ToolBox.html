desc

A RedBeanPHP-wide service locator
author

license

BSD/GPLv2 ToolBox. The toolbox is an integral part of RedBeanPHP providing the basic architectural building blocks to manager objects, helpers and additional tools like plugins. A toolbox contains the three core components of RedBeanPHP: the adapter, the query writer and the core functionality of RedBeanPHP in OODB. copyright (c) G.J.G.T.
